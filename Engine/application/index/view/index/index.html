关于人才库

“专业人才库管理中心”下设24种大类，840种类别专业的人才培养。 目前已成立了易学国学、医疗美容、健康教育、少儿艺术、婴幼儿早期教育、机器人技术、建筑设计国际认证、管理创新教育、在线教育、防水工程、艺术类培训等项目，主要以传承中国优秀传统文化、倡导健康和美化人民生活、提高中国下一代和全民素质、建立和谐社会、鼓励创新创业为主要发展方向，并且设立全国专业人才考评专家委员会和项目专家委员会，以专业化为主导，由行业内最具权威的国内、国外专家进行专业化的指导，树立行业标准，为中国和世界进行人才的交流搭建平台。
